1. There is one external library used, for that run "sudo apt-get install libreadline-dev".
2. Run Makefile by giving run command.
3. getinput.c is the file which helps in taking input.
4. commands.c id the file which helps in cd,pwd and echo.
5  ls.c helps in ls command.
6. pinfo.c is the file which will call pinfo function.
7. execute.c is the file which helps in background and foreground process.
8. his.c is the file which helps in keeping history.
